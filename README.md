# Console Tic-Tac-Toe

A simple implementation of the game for educational purposes.

## Play

A 3x3 grid will be shown.
Each player will input the desired cell number where to place their marker.

```
|---|---|---|
| 1 | 2 | 3 |
|---|---|---|
| 4 | 5 | 6 |
|---|---|---|
| 7 | 8 | 9 |
|---|---|---|
Enter cell number: 
```
